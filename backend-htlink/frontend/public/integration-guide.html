🏨 Hotel Analytics Tracking Integration

Complete guide to integrate analytics tracking into app.botonblue.com

📋 Overview

This system tracks visitor behavior on your hotel website and provides detailed analytics through a dashboard. Here's what gets tracked:

  • Page Views: Every page visit with device detection
  • User Interactions: Clicks, form submissions, scroll behavior
  • Session Data: Time on site, bounce rate, user flow
  • Real-time Metrics: Active users, current activity

Step 1: Add Tracking Script to Your Website

Add this script to the <head> section of all pages on app.botonblue.com:

<!-- Hotel Analytics Tracking Script --> <script> (function() { 'use strict'; // Configuration for Boton Blue Resort const CONFIG = { API_BASE_URL: 'http://localhost:8000/api/v1/analytics', TRACKING_KEY: 'botonblue-tracking-key', DEBUG: false // Set to true for development }; // Utility functions const log = (...args) => { if (CONFIG.DEBUG) console.log('[Analytics]', ...args); }; const getDeviceType = () => { const width = window.innerWidth; if (width <= 768) return 'mobile'; if (width <= 1024) return 'tablet'; return 'desktop'; }; const generateSessionId = () => { return 'session_' + Date.now() + '_' + Math.random().toString(36).substr(2, 9); }; const getOrCreateSessionId = () => { let sessionId = sessionStorage.getItem('hotel_analytics_session'); if (!sessionId) { sessionId = generateSessionId(); sessionStorage.setItem('hotel_analytics_session', sessionId); } return sessionId; }; // Main tracking function const trackEvent = async (eventType, additionalData = {}) => { try { const eventData = { tracking_key: CONFIG.TRACKING_KEY, event_type: eventType, device: getDeviceType(), user_agent: navigator.userAgent, url: window.location.href, referrer: document.referrer || undefined, page_title: document.title, session_id: getOrCreateSessionId(), timestamp: new Date().toISOString(), ...additionalData }; log('Tracking:', eventType, eventData); const response = await fetch(`${CONFIG.API_BASE_URL}/track`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(eventData) }); if (response.ok) { const result = await response.json(); log('Tracked successfully:', result); return result; } } catch (error) { console.error('Analytics error:', error); } }; // Auto-track page view const trackPageView = () => { trackEvent('page_view', { viewport_width: window.innerWidth, viewport_height: window.innerHeight, screen_width: screen.width, screen_height: screen.height, timezone: Intl.DateTimeFormat().resolvedOptions().timeZone, language: navigator.language }); }; // Track clicks on important elements document.addEventListener('click', (event) => { const element = event.target.closest('a, button'); if (element) { trackEvent('click', { element_tag: element.tagName.toLowerCase(), element_text: element.textContent?.trim().substring(0, 100) || '', element_href: element.href || '', element_id: element.id || '', element_class: element.className || '' }); } }); // Track form submissions document.addEventListener('submit', (event) => { const form = event.target; if (form.tagName === 'FORM') { trackEvent('form_submit', { form_id: form.id || '', form_action: form.action || '', form_method: form.method || 'get' }); } }); // Initialize tracking when DOM is ready if (document.readyState === 'loading') { document.addEventListener('DOMContentLoaded', trackPageView); } else { trackPageView(); } // Expose global tracking function window.HotelAnalytics = { track: trackEvent }; })(); </script>
⚠️ Important: Make sure to update the API_BASE_URL to your production server URL when deploying to production!

📊 Event Types Tracked

Event Type Description Automatic/Manual
page_view User visits a page Automatic
click User clicks links or buttons Automatic
form_submit User submits a form Automatic
booking_start User starts booking process Manual
booking_complete User completes booking Manual

🛠️ Advanced Custom Tracking

For specific hotel events, you can add manual tracking:

Booking Events:

// When user starts booking HotelAnalytics.track('booking_start', { room_type: 'deluxe_suite', check_in: '2024-12-01', check_out: '2024-12-05', guests: 2 }); // When booking is completed HotelAnalytics.track('booking_complete', { booking_id: 'BK12345', total_amount: 1200, currency: 'USD', payment_method: 'credit_card' });

Room/Gallery Interactions:

// When user views room details HotelAnalytics.track('room_view', { room_id: 'deluxe_101', room_type: 'Deluxe Ocean View' }); // When user opens photo gallery HotelAnalytics.track('gallery_view', { gallery_type: 'room_photos', photo_count: 15 });
